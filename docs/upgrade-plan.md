# 浮力大發現 — 升級執行計劃書
> 版本：v1.0 | 日期：2026-05-26 | 目標：香港教育局示範展示

## 🎯 升級願景

一款讓SEN小學生通過觸碰、觀察與遊戲，理解「浮力與密度」科學概念的互動實驗平台。
在iPad、Smartboard上呈現專業但可愛的體驗。

**核心感官印象：** 寫實物品 × 可愛介面 × 精準科學可視化

## ⚙️ 三、物理機制升級

### 3.1 連續密度引擎（核心）

```javascript
// 物品屬性重構
const item = {
  name: '木塊',
  image: 'assets/images/wood-block.png',
  density: 0.6,        // 連續密度值 g/cm³
  floats: true,         // 預測結果
  reason: '木頭密度比水低',
  senHint: '木塊好輕，會浮係水面',
  label: '密度: 0.6 g/cm³'
}

// 水屬性（可調節）
const water = {
  density: 1.0,        // 清水
  // density: 1.03,    // 鹽水（部分物品會改變結果）
  // density: 0.9,     // 油
}
```

### 3.2 視覺下沉/漂浮動畫

```
目前：binary 即時切換
升級：基於密度差，慢慢過渡
- 木塊（0.6）：2秒內慢慢浮到水面，停住
- 石頭（2.5）：1.5秒慢慢沉到水底
- 冰塊（0.9）：接近水面，停在0.9位置
```

### 3.3 阿基米德原理視覺化

**排水量顯示：**
- 物品放入水池時，排水範圍藍色高亮
- 旁邊即時顯示：`排水量：XX ml`
- 天秤比較模式：兩件物品並排，show邊個排更多水
